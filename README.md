本项目是一套pc端通过adb驱动usb演示控制手机的java项目源码，目前可以实现手机屏幕的实时演示，效果跟360手机助手pc版差不多。另外实现了对手机HOME、BACK、MENU三个按键的控制。不过在我的摩托古董机(2.3.3系统)上貌似不管用。另外如果你的手机分辨率很大的话会演示范围会默认自动超出屏幕。可以点击上方菜单里的缩小比例即可。另外需要注意的是项目导入eclipse的时候选择“现有项目到当前工作空间中”，导入以后直接在com.lenovo.ScreenCapture包下面的MainWindow.java文件上右键运行即可。
